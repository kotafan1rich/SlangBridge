# SlangBridge

Проограмма для перевода слов с различного сленга.

---

### Возможности
* Перевод слова со сленга
* Перевод предложения/словосочетания с использованием ии
* Оффлайн словарь часто используемых слов

### Использование
* Скачать последний релиз  
* Распаковать архив
* Запустить .exe файл
